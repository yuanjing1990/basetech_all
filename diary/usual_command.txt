#发送邮件
sudo apt install s-nail
vi /etc/s-nail.rc
添加：
set from=邮箱地址 smtp=smtp.邮箱地址后缀 //如set from=123456@qq.com smtp=smtp.qq.com
set smtp-auth-user=邮箱地址 smtp-auth-password="邮箱密码" smtp-auth=login
配置好后就可以发送邮件了:
s-nail -s "主题" "目的邮箱地址" <<< "邮件内容"
